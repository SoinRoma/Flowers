<h1 align="center">Flowers-Shop</h1>

## Описание
Интернет магазин по продаже цветов.
На сайте реализовано:
+ Слайдеры
+ Магазин для выбора цветов
+ Адаптивная верстка
+ Bootstrap4
+ Модальные окна
+ Fancybox
